# Daisy 1.0.6.4 — open source + UX polish

## Changed

- **Daisy is now fully open source under Apache 2.0.** Switched from BUSL 1.1 — no commercial-use carve-outs, no time-bombed conversion clock, standard OSI-approved permissive licence.
- **Voice notes read as voice notes.** Sessions recorded via the voice-notes hotkey were titled "Meeting yyyy-mm-dd hh:mm" regardless of mode. Title now reflects the actual recording mode — "Voice note …", "Dictation …", "Meeting …".
- **Cleaner Library + session header.** Removed the AUTO/EN/RU locale chip, the system-audio speaker icon (in both session header and Library list rows), and other redundant decorations. List rows now show only summary + screenshots badges; header is date · duration · tag.
- **Settings → Profile.** Renamed the "You" section to "Profile" to match the macOS Settings convention.
