# Daisy 1.0.7.13 — a smaller widget and a safer quit

## New

- The floating widget is a bit smaller now, so it sits more quietly out of your way while you work.
- If you quit Daisy while it's still recording, it now asks you first — and if you confirm, it finalizes and saves the recording (audio and the transcript so far) before closing, instead of losing it.

## Fixed

- "Draft follow-up" now always writes a follow-up when you ask for it, even for a conversation Daisy would otherwise treat as internal.
- Recording from your built-in mic no longer goes silent when you're playing audio through Bluetooth headphones — Daisy now avoids the headset's low-quality Bluetooth mic from the start of a recording (it already switched away from it mid-recording).
- Long transcript lines no longer spill past the edge of their card.
