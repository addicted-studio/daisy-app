# Daisy 1.0.7.14 — choose your dictation engine

## New

- You can now pick which on-device model powers **dictation** in **Settings → Transcription → Dictation engine**: **Whisper** (default) or the new, experimental **Parakeet** — a faster streaming model that covers 25 European languages including Russian. Switching to Parakeet downloads its model once (~600 MB) with a visible progress bar. Meetings always use Whisper, and if Parakeet can't run, dictation falls back to Whisper automatically.
