# Daisy 1.0.7.22

- **Long transcripts no longer get cut off.** A long meeting (40+ minutes) could render only partway in the Library with no way to scroll to the rest. The transcript now lives in its own scrollable pane, so you can read — and select — all of it, however long the recording.
- **Compact menu bar, reworked.** With "Compact menu bar" on (Settings → General → Appearance), clicking the menu-bar icon now opens a quick menu — Summarize, Open Library, Settings, New recording, Quit — instead of the full transcription window. It no longer hides your Dock icon or the app menus.
- **Calmer Record button, with the shortcut on it.** The Record button in the menu-bar popover is now the same warm beige as the main window (no more green), and the main window's Record button shows your record shortcut right on it.
- **Dictation page tidy.** "Add word" and "Clear history" moved to the top-right of their sections as quick capsule buttons.
- **Voice Memos import: pick where it saves.** If you import Apple Voice Memos, you can now choose the exact folder the transcripts land in — separate from your meeting sessions — so they drop straight into your notes/vault.
- **Summary, Follow-up and Transcript are now three separate panels**, each with its own copy button — the follow-up draft got pulled out into its own section. The Summarize button now shows a spinner while it's working.
- Fixes: the Record button no longer briefly flashes a red "Try again" right after a cold launch; the top of the window no longer shows a white strip in light system mode; and a meeting that's too long for Apple Intelligence's on-device model now tells you to switch to a cloud provider instead of failing silently.
- Smaller polish: the status line no longer repeats what the Record button already says, the Library toolbar drops a redundant Copy button (every copy option still lives in the ⋯ menu), and the automatic-update control is now a switch.
