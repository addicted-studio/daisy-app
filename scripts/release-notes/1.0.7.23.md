# Daisy 1.0.7.23

- **Copy any one part of a session.** The ⋯ menu in the Library now has **Copy Summary**, **Copy Follow-up** and **Copy Transcript** — each copies just that section as Markdown — plus **Copy all** for the whole thing. (The old "Copy as Markdown / Copy for Obsidian" pair is gone: the only difference between them was a metadata header, and Daisy's routing already handles vault filing.)
- **The dictation flower reacts to your voice again.** While dictating, the petals barely moved as you spoke; they now respond with the same sensitivity as meeting recording.
- Smaller polish: the ⋯ button in the Library toolbar is now a proper rounded capsule instead of a tight square.
