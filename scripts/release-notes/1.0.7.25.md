# Daisy 1.0.7.25

- The floating widget's "Hide for…" menu now has a real **Hide for 24 hours** — the old "Hide for the day" actually brought the widget back after 8 hours. Hiding it now also sticks if you quit and reopen Daisy, instead of reappearing on the next launch.
- The **Dictation** page now uses the same native tabs as Settings, and **Add word** is available from both the Vocabulary and History tabs (it used to disappear on History).
- **Voice Memos import** moved to Settings → Transcription and got a tidier layout: the save folder and its "Choose folder…" picker now share one row, and **Process existing recordings** shows live progress and tells you how many it imported when it's done.
- The Full Disk Access request for Voice Memos now lives in Settings → Permissions, clearly marked as needed only for Voice Memos.
