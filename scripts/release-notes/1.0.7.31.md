# Daisy 1.0.7.31 — new Home, Notes, and Voice

## Added

- **Notes are now a first-class type.** A dedicated Notes tab sits alongside your recordings, and both share the same projects and tags — so a note and a meeting can live under the same project and be filtered the same way.
- **Voice profile.** Daisy can build a profile of how you write from your own dictations and, optionally, polish each dictation to sound like you before it's pasted. You can also paste or import a style you already have from another app and edit it in place.
- **Redesigned Home.** A single "your day" card merges the brief, your agenda, and open tasks (with overdue and completed grouped separately), plus an activity heatmap and a setup checklist.
- **Report a bug from About.** A new "Report a bug" button in the About section sends a log report the same way the Help menu does.
- **Guided setup goes further.** After permissions, first-run now offers optional steps to pick your recordings folder, connect your calendar for auto-recording, choose the AI that writes summaries, and teach Daisy dictation terms or a writing-style prompt — each fully skippable.

## Changed

- **Library** moves to a cleaner three-column layout with filtering by project and tag.
- Refreshed typography and a calmer, more consistent look across Home, Library, and the Voice section.
- Russian localization pass across the Home screen and the Voice section.

## Fixed

- **Summary generation with LM Studio** no longer fails. Newer LM Studio builds reject the older JSON request mode; Daisy now works across LM Studio versions.
- Empty recordings now say so instead of showing a blank transcript panel.
- The day brief no longer occasionally came up blank on launch before your calendar and tasks had finished loading.
