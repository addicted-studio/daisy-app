# Daisy 1.0.7.35 — update badge in the sidebar

## Added

- **A quiet "Update" button in the sidebar.** When a newer version of Daisy is available, an Update action now appears at the bottom of the side menu — one click starts the install. Unlike the update prompt, it stays put as a gentle reminder if you're not ready yet, and clears itself once you're on the latest version. Daisy does a silent check shortly after launch so it shows up promptly, and only when you have automatic update checks enabled.
