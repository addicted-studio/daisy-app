# Daisy 1.0.7.6 — recordings never silently lost, calendar picker fixed, bigger widget

## Fixed

- **A recording is never silently discarded on Stop (build 48).** If a session ends with no captured audio and no transcript — for example when live transcription produced nothing — Daisy now keeps the recording and tells you, instead of quietly removing it. A genuinely empty, very short accidental tap still cleans itself up, but with a notice rather than in silence.
- **Calendar picker now matches the menu bar (build 48).** The menu-bar "next meeting" label and the recording popover's calendar picker now use the same look-ahead window, so an event the menu bar shows is always selectable in the picker. Previously an event roughly 6–8 hours out appeared in the menu bar but the picker reported "No upcoming events."

## Changed

- **Bigger recording widget (build 48).** The floating daisy widget and its panel are about 10% larger.
